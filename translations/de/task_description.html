In der Mathematik, insbesondere der linearen Algebra, ist eine schiefsymmetrische Matrix (auch antisymmetrische Matrix) eine quadratische Matrix A, deren Transponierte gleich ihrem Negativen ist.
Dies bedeutet, dass sie die Gleichung A = −ATerfüllt.
Wenn also der Eintrag in der i-ten Zeile und der j-ten Spalte aij ist, z. B. A = (aij), dann gilt aij = -aji.

Du sollst feststellen, ob die gegebene quadratische Matrix schiefsymmetrisch ist oder nicht.

Weitere Informationen zu schiefsymmetrischen Matrizen findest Du z. B. auf der entsprechenden Wikipediaseite .

Eingabe: Eine quadratische Matrix als Liste von Listen mit ganzen Zahlen (int).

Ausgabe: Ob die Matrix schiefsymmetrisch ist oder nicht, als boolescher Wert (True oder False).

Beispiele: 

checkio([
    [ 0,  1,  2],
    [-1,  0,  1],
    [-2, -1,  0]]) == True
checkio([
    [ 0,  1, 2],
    [-1,  1, 1],
    [-2, -1, 0]]) == False
checkio([
    [ 0,  1, 2],
    [-1,  0, 1],
    [-3, -1, 0]]) == False

Wozu das gut ist:
Matrizen sind ein Schlüsselkonzept für die lineare Algebra und Vektorgrafiken.
Schiefsymmetrische Matrizen können für das Bilden des Kreuzproduktes nützlich sein, eine Operation in der Mathematik, die bei der Berechnung der Bewegung von Kräften verwendet wird.

Bedingungen:
0 < N < 5
