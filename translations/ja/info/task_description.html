数学では、特に線形代数では、スキュー対称行列(skew-symmetric matrix, antisymmetricまたはantimetricとしても知られています)は、その転置行列が 負であるような正方行列です。 これは、それが方程式 A = −AT を満足していることを意味しています。 i行j列のエントリーをaijとすると、すなわちA = (aij) とすると、スキュー対称となる条件は aij = −aji となります。

指定された正方行列がスキュー対称かどうかを決定しなければいけません。

スキュー対称行列の詳細についてはWikipediaのページで詳細を見れます。 (訳注: 日本語では反対称行列とか交代行列と呼ばれています。日本語のwikipediaページ)

入力: 正方行列、整数のリストのリスト

出力: この行列がスキュー対称であるかどうか、ブール値

例題: 

checkio([
    [ 0,  1,  2],
    [-1,  0,  1],
    [-2, -1,  0]]) == True
checkio([
    [ 0,  1, 2],
    [-1,  1, 1],
    [-2, -1, 0]]) == False
checkio([
    [ 0,  1, 2],
    [-1,  0, 1],
    [-3, -1, 0]]) == False

どのように使われるか: スキュー対称行列は、力の移動の計算に使う数学操作である直積に役に立つ。 行列は線形代数とベクトルグラフィックの基礎である。

事前条件: 0 < N < 5
